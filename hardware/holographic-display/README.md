# Holographic Display

Desktop Pepper's Ghost holographic display. Materials: 4" LCD, acrylic pyramid (51mm base, 45-degree angle, 57mm slant height), WS2812B LED strip, Arduino/ESP32. Cost: $96-177. Build time: 7-13 hours. Power: ~15W. Python code for 4-quadrant video generation (OpenCV). Arduino C++ for LED effects (breathing, rainbow, reactive).
