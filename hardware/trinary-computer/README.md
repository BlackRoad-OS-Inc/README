# Trinary Computer

Physical ternary computing hardware. SVG laser-cutting templates for 6mm plywood/acrylic. 4x4 grid of trinary switches (+1, 0, -1) with brass rod actuators and magnet detents. Base plate specs, registration marks, mounting holes. Connects to base-3 optimality research (37% improvement over binary).
