# Jetbrains Mono

JetBrains Mono — code and monospace font. Weights: 400 (code), 500 (micro labels). Size: 11-13px (code), 9-10px (timestamps, micro). Letter-spacing: 0.06-0.15em for uppercase labels. Used for: code blocks, terminal, section labels, badges, stats, timestamps.
